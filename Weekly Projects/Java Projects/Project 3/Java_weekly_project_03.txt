Απαιτείται να υλοποιήσετε:
1. Ο παίκτης παίζει ένα δελτίο Τζόκερ (δίνει τους αριθμούς)
2. Το κάθε δελτίο Τζόκερ πρέπει να έχει διαφορετικό ID
3. Γίνεται κλήρωση (5 τυχαίοι αριθμοί και ένας Τζόκερ)

4. Έλεγχος αν ένα δελτίο νίκησε. Νικητήρια δελτία είναι αυτά που έχουν
    α. 5+1 επιτυχίες (όλα σωστά)
    β. 4 επιτυχίες (4 αριθμοί μέσα στους 5 που κληρώθηκαν)

5.  Κάντε 20 κληρώσεις υπολογίστε τους 
    α. Τους 3 περισσότερο κληρωθέντες αριθμούς
    β. Τους 3 λιγότερο κληρωθέντες αριθμούς
